Aplicativo mobile simples feito com React Native (Expo) para cadastrar dados de um aluno.

O usuário preenche nome, curso, disciplina e descrição pessoal; ao enviar, o app valida os campos e mostra as informações na tela.

## Rodar o projeto

1. `npm install`
2. `npm start`
3. Caso esteja usando Expo App, escanear o QR Code usando a mesma rede.
4. Caso use Android Studio, pressione `a` no terminal.
